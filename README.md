## 📖 Overview

Coffee Project is a modern web application designed for coffee lovers. It provides an elegant user interface to explore coffee products, learn about different coffee varieties, and enjoy a seamless browsing experience.

---

## ✨ Features

- 🎨 Modern & Responsive Design
- 📱 Mobile-Friendly Interface
- ☕ Coffee Product Showcase
- 🔍 Easy Navigation
- ⚡ Fast Performance
- 🌙 Clean User Experience
- 📍 Contact & About Sections

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/kpaushal/coffee-project.git
```

### Navigate to Project

```bash
cd coffee-project
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Application will run at:

```bash
http://localhost:5173
```

---

## 🔧 Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
